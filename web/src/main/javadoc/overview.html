This is a web application using AngularJS on the client side and SpringMVC for REST API on the server side.

The REST API conforms to HATEOAS principles and uses HAL (Hypertext Application Language) for JSON rendering. See

  • Spring HATEOAS
  • Hypertext Application Language (HAL)
  • Hypertext Application Language (Wikipedia)

The AngularJS interface consists of the src/mainwebapp/index.jsp main page, which uses HTML templates from the src/mainwebapp/partials directory.
